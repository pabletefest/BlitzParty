# BlitzParty [![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

<img width="512" height="512" alt="imagen" src="https://github.com/user-attachments/assets/69bbcacf-38a2-4ab9-882c-464ee3917d2b" />

The project, titled Blitz Party, is a multiplayer video game for mobile devices that belongs to the
genre of games based on minigames. It is a free-to-play application that includes extra content
that can be purchased with microtransactions. It also contains offline features and character
customization, among many others.

The project has been developed by a team of four students (Swifter Games),
within the framework of the development of their Bachelor's Degree Final Project. The game was developed with Unity Engine and C# programming language.
